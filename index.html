Don't worry - you're not alone.

Ruby + Ruby on Rails

Ruby is a favorite of web-focused developers, especially those who are interested in building applications that users can interact with. AirBNB, Groupon, and Hulu are just a few examples of popular websites built with Ruby.

Ruby is very often used in cojunction with a framework called "Ruby on Rails", or "Rails" for short. Rails is basically a collection of pre-built functions (think of them as miniature programs), which simplify a lot of the most common tasks a Ruby developer might need to do.

Learn more about Ruby and Rails. You can also preview the Ruby coursework and the Rails coursework.

PHP + Drupal

PHP is - by far - the most popular "backend" (server-side) language in use today. Developers use it to collect user input, generate dynamic on-page content, and more.

PHP is very commonly used alongside content management systems like Drupal, WordPress, and Joomla. After gaining a strong PHP background, Epicodus students will learn to use advanced tools found within the Drupal framework and become experts in building and maintaining content-heavy websites.

Learn more about PHP and Drupal. You can also preview the PHP coursework and the Drupal coursework.

Java + Android

Java might just be the most popular programming language in use today. It's used for everything from web and mobile apps to high-performance enterprise software. Java has long history of use in many major applications - the underlying language is actually more than two decades old!

Java is also the official language of Android app development. Since Android is currently the most popular operating system out there, it's no surprise that capable Java and Android developers are still in very high demand by businesses large and small.

Learn more about Java and Android. You can also preview the Java coursework and the Android coursework.

CSS + Web Design

CSS is the language used to style webpages. Developers and designers use CSS to quickly change layouts, colors, fonts, and more across entire webpages or websites. It can also help adapt your websites to display properly on all sorts of different devices and screen sizes.

No matter what industry you work in, it's important for your websites to be visually pleasant and easy for users to interact with. After all, your product or service's success depends on it!

Learn more about CSS and Web Design. You can also preview the CSS coursework and the Web Design coursework.

C# + .NET

C# (pronounced as "see sharp") is a flexible language used to develop everything from desktop software to mobile apps to video games. And because it's been around for a long time and has the backing of Microsoft, it's one of the most in-demand languages - especially by large, established corporations and government agencies.

Epicodus students also learn to use the .NET framework, which is basically a collection of pre-built functions (think of them as miniature programs) that simplify a lot of the most common tasks a C# developer might need to do. .NET is particularly useful for building web applications with C# - some popular websites built on .NET include Microsoft's own site, Bing, and Stack Overflow.

Learn more about C# and .NET. You can also preview the C# coursework and the .NET coursework.
